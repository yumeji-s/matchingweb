Overview

Nikoichi(ニコイチ)は学生が作った完全無料のマッチングアプリ！！！ 自分の趣味を登録して好きなことで繋がろう！！！！！

Feature

  • 機能1. アカウントを登録
  • 機能2. プロフィール、好きなこと、やりたいことの登録
  • 機能3. 自分と趣味の合う、やりたいことが同じなどのおすすめの人表示機能
  • 機能4. 気になる人がいたら「気になる！」を送る機能
  • 機能5. 送られた「気になる！」を表示して「気になる！」を送り返す機能、マッチング成立機能
  • 機能6. マッチングした人を表示してチャットができる機能
  • 機能7. やりたいことなどで同じ趣味の人を検索できる機能
  • 機能8. 他のアプリやサービスのアカウントと連携できる機能

Background

自分の実力を確認するために、どの程度の開発案件ができるのか挑戦したかったので、色々なサイトで案件を調べて回った。

その結果見つかったのがマッチングアプリの開発案件だった！価格帯は300000〜500000円くらいだった。

様々な環境での開発の仕方を学ぶため、今自分が熱いと思っているサービスやフレームワークを使って開発を進めようと思った。

また、学習に重きを置いているため、細かなことでも気になったら派生させて調べ、面白いツールなどがあったら導入していこうと思う。

What I Use

  • 使用言語はJavaScriptで、ReactNativeというフレームワークでアプリ開発を行います。

  • firebaseというプラットフォームで、webサイトの公開や、リアルタイムDBの実現、ログインや認証まわりの工数削減を狙います。

  • githubでバージョン管理、家と学校などの複数箇所での開発を楽に行います。

  • laravelは就職先でも使うため、webサイト構築に用います。

  • Herokはlaravelで作ったwebサイトを公開するために使います。

  • github Pagesは調べていて気になったので、webサイト公開に用います。

  • jQueryは一度勉強しておきたかったので使います。

  • CircleCIで、githubとそれぞれのホスティングサービスとを連携させてビルドやテスト、デプロイまでを自動化します。
